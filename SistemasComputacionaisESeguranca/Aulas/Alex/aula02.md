###### Operações aritmética combinadas 

   
    Soma:
    1 + 1 = 0 e vai 1
    0 + 1 = 1
    1 + 0 = 1
    0 + 0 = 0
	_________________________
	Subtração:
    0 - 0 = 0
    1 - 0 = 1
    1 - 1 = 0
    0 - 1 = 1 e vai 1
    0-1-1 = 0 e vai 1
    1-1-1 = 1 e vai 1
